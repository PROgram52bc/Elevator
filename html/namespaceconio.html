◆ gotoRowCol()

string conio::gotoRowCol ( const int  row,
const int  col 
)

Positions cursor to the specified row, col location.

Gotoxy will position the cursor at the specified row,col location. The upper left corner has the coordinates 1,1. Note: this is row,col, not x,y (col,row) as is often done.

Parameters
rowThe row coordinate (1-based).
colThe column coordinate (1-based).
Returns
Returns a string containing the escape sequence to send to the screen. gotoRowCol = CSI r;c
